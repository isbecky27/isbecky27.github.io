荷比自由行 - Day1 鹿特丹

July 7, 2025 · 2 min · 214 words · Me

Day1 路線圖#

萊登 Leiden Centraal ➜ 鹿特丹 Rotterdam Blaak#

在萊登和朋友拿了荷蘭博物館卡 aka. 尿尿神卡後，就搭車前往鹿特丹了 ~
( 現在有數位卡了 ~ 但我們想做紀念，所以還是選了實體卡 XD )

萊登 Leiden Centraal 到鹿特丹 Rotterdam Blaak 車程約 40 min，買了荷鐵早鳥票 NS PriceTime Deal €5.64 ( 原價 €9.40，直接打了 6 折 )！荷鐵的早鳥票很有誠意，不綁車次，只限制搭乘時間，時間範圍也給的蠻大方的 ~

附上萊登火車站

鹿特丹的著名景點都在 Rotterdam Blaak 站 ! 不是 Rotterdam Centraal 歐 !

Check In - 鹿特丹世民酒店#

火車站出來對面就是這次住的飯店了 — citizenM Rotterdam 鹿特丹世民酒店

( 飯店門口處旁邊就有個火車站出入口，只是沒手扶梯 )

先講優點

  • 地點非常好，離景點和火車站都超近
  • 服務人員很熱情，自動 Check In / Out 很快速
  • 房間風景很美 ( 入住時可挑選想要面那個風景的房間，好像有面像方塊屋的 )
  • 免費租借腳踏車

缺點

  • 沒有補備品 ( 洗髮乳/沐浴乳全空 )
  • 浴室排水口沒清理，洗澡直接大淹水
  • 冷氣沒有作用很熱 ( 服務人員告知是因房間面向的位置會照到太陽，所以冷卻較慢，但過了整晚，早上起床房間依舊很熱 )
  • 房間蠻小的，開兩個 28 吋行李箱有點困難

剛下飛機想洗澡，但因為沒有洗髮精 + 浴室淹水無法洗澡.. 告知服務人員後，等了半小時都還沒有人來處理，只好作罷先出門晃晃 ~

BTW 飯店旁的舊港 Oude Haven ( 我們房間面的 View ) 很漂亮！緊鄰方塊屋，可以順道晃晃 ~

“在荷蘭騎腳踏車” 成就解鎖失敗

和飯店租了腳踏車，滿心期待要解鎖「在荷蘭騎腳踏車」的成就，結果我根本沒辦法騎呀 QAQ

荷蘭的腳踏車真的是太高啦，幾乎踩不到地阿 ~ 兩隻腳都踮到一個極限 Be Like:

而且踏板操作方式也不一樣，往後踩是煞車。還有研究腳踏車鎖也是花了一陣子 ᶘ ᵒᴥᵒᶅ 為了安全著想最後還是放棄腳踏車了：）

鹿特丹市區晃晃#

路線：方塊屋 ➜ 鉛筆屋 ➜ 拱廊市場 ➜ 聖勞倫斯大教堂 ➜ 市政廳 ➜ 買晚餐 & 超市採購

走出飯店，基本上一眼就能看完鹿特丹最有名的幾個景點啦 XD

從左到右依序是：拱廊市場、Rotterdam Blaak 車站、水管圖書館、鉛筆屋、方塊屋 （ 照片這個角度的正後方，就是我們住的飯店！）

方塊屋 Kubuswoningen
方塊屋有開放參觀，但之前看影片覺得還好就沒花錢進去了 ~ 如果想體驗住方塊屋的話，可以住一晚 Stayokay Hostel Rotterdam。 剛好遇到一團台灣旅遊團，導遊說從正下方往上拍，會剛好拍出六角星，於是就有了右邊這張照片 XD

鉛筆屋 & 水管圖書館

拱廊市場 Markthal
拱廊市場的亮點就是它的外觀 & 室內天花板上的蔬菜水果 (?) 很像台灣日曆上會出現的圖案 XD
市場裡面賣很多異國料理 ( 日式、中式都有 )，才剛到歐洲比較想吃歐式料理，所以就只有簡單晃過 ~ 傍晚會多顆愛心 ~

聖勞倫斯大教堂 Grote of Sint-Laurenskerk
持博物館卡可免費參觀，不過我們抵達的時候它已經關門惹 QAQ 只能在草皮上休息吃冰淇淋拍照

號稱世界最美的麥當勞🍟
嘿，我覺得還好 XD 不過剛好路過還是順手拍張照 ~

鹿特丹市政廳 Stadhuis van Rotterdam
路過踩個點 XD 前面的馬路有夠大條 ( X )

晚餐 Dinner 🍽️
買了 Pizza 回飯店吃，還有去超市買了點水果 ~

btw 草莓跟藍莓有夠便宜又有夠好吃 (❁´◡`❁) 附上夜晚的飯店

總結#

鹿特丹給人的第一印象就是「非常現代」，建築設計感十足，不過也因為這樣少了點歐洲的味道（？）比起觀光，這裡可能更適合生活。搭火車前往機場、阿姆斯特丹或是比利時都很方便，可以當作旅行的中繼站。

至於停留時間.. 如果只想走走前面提到的幾個景點，半天可能就差不多了 XD 但如果想順道走訪周邊城市 ( 海牙、代爾夫特 )，或是前往稍遠一點的小孩堤防，那會建議多安排個 1.5 天比較充裕！
其他的行程細節，就繼續看接下來兩天的遊記吧 ~ (๑•̀ㅂ•́)و✧

順帶一提，不知道是這裡水太硬還是空氣太乾躁，手超乾超不舒服，擦了乳液還是一樣，第一次感受到水土不服 இ௰இ
